# Jmeter component

Jmc Classes use the some name of Jmeter class with 'wrapper suffix'.

## Thread Group

All Thread group implements interface **ThreadWrapper**.

| Component                 | covered | class                   | Description      |
| ------------------------- | ------- | ----------------------- | ---------------- |
| **Thread Group**          | **yes** | ThreadGroupWrapper      | **Full support** |
| **setUp Thread Group**    | **yes** | SetupThreadGroupWrapper | **Full support** |
| **tearDown Thread Group** | **yes** | PostThreadGroupWrapper  | **Full support** |
